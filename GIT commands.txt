1. git init - иницилизация
2. git status - Проверка статуса
3. git config user.name 'name' - Обязательно добавить юзернейм
4. git config user.email 'enter email' - Обязательно добавить почту
5. git add . - Взять все файлы
6. git commit -m 'название' - создание репозитория(хранилище для хранения данных)
